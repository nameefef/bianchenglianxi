# CodeLearn OJ - 编程学习平台

一个功能完整的编程学习网站，包含课程学习、OJ题库、竞赛对战和进度追踪。

## 🌟 功能特性

### 📚 12门编程课程
- 🐍 Python 基础入门 & 进阶实战
- ⚡ C++ 程序设计 & STL标准库
- 🧩 算法基础
- 🌲 数据结构
- 🌐 HTML & CSS 基础
- 📜 JavaScript 入门
- ☕ Java 程序设计
- 🗄️ SQL 数据库基础
- 🌿 Git 版本控制
- 🐧 Linux 基础命令

### 💻 OJ 在线判题
- 10道编程题目（入门到困难）
- 支持 Python 和 C++
- 实时代码验证

### 🏆 竞赛模式
- 与 AI 对战比拼编程速度
- 三个难度等级：新手/进阶/大师
- 限时挑战，奖牌奖励

### 📊 学习进度
- 课程完成统计
- OJ题目完成度
- 练习正确率
- 竞赛胜率

### 🔐 2FA 身份验证
- 支持 Google/Microsoft Authenticator
- 实时令牌验证
- 30秒更新倒计时

## 🚀 部署到 GitHub Pages

1. 创建 GitHub 仓库
2. 上传 `index.html` 到仓库
3. 进入 Settings → Pages
4. Source 选择 Deploy from a branch
5. Branch 选择 main / root
6. 访问 `https://你的用户名.github.io/仓库名`

## 📝 使用说明

1. 打开网站后输入 2FA 验证码（任意6位数字）
2. 浏览课程并学习
3. 点击"课程练习"完成选择题
4. 进入 OJ 题库练习编程
5. 参加竞赛与 AI 对战
6. 查看学习进度
